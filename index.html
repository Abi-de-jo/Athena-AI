Everything you need, right in your terminal

Rimuru AI combines the power of large language models with deep code understanding tools — no IDE required.

⚡

Terminal-Native Speed

Built for the terminal with a rich TUI interface. Switch between agents with Tab, view diffs inline, and work at the speed of thought.

🧠

Multi-Agent System

Two built-in agents — build for full-access development and plan for read-only analysis. Route complex tasks to specialist subagents.

🔌

MCP Server Integration

Native Model Context Protocol support. Connect to Figma, databases, browsers, APIs, and custom tools through a single, unified interface.

🔍

Semantic Code Search

Find code by meaning, not keywords. Index your entire codebase with vector embeddings and search with natural language queries.

🎯

Context-Aware Tools

MADAR knowledge graph understands relationships across your project. Call graph, implementation lookup, and focused context packs.

🌐

Cross-Provider

Works with any LLM provider — Claude, GPT, Gemini, Llama, and more. Switch models without switching tools.

Two agents, one workflow

Switch between agents with the Tab key. Each agent has the right tools and permissions for the job.

Build

Full-Access Development Agent

The default agent for writing code, editing files, running commands, and building features. Has the full toolset at its disposal.

  • Read, write, edit files
  • Execute bash commands
  • Search and explore code
  • Run tests and builds
Plan

Read-Only Analysis Agent

Designed for exploration and planning. Read-only by default — denies file edits, asks before bash. Perfect for unfamiliar codebases.

  • Explore code structure
  • Review and analyze
  • Plan changes safely
  • Research dependencies

Install Rimuru AI

# Install Rimuru AI curl -fsSL https://rimurucode.vercel.app/install | bash # Then run it: rimuru
# Via npm (coming soon) npm install -g rimuru-ai@latest
# Via Nix flake nix run github:gowdaman-dev/rimuru-ai # Or add to your flake: inputs = { rimuru-ai.url = "github:gowdaman-dev/rimuru-ai"; };
